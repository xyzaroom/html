## Xyzaroom Records Inc.

A modern, responsive digital music label website. Cloned & enhanced from Red Music Digital.

## Features
- Swiper.js hero slider (via CDN)
- Responsive grid layout
- Smooth scroll navigation
- Dark theme with red accent
- Ready for GitHub Pages

## Tech Stack
- HTML5, CSS3, Vanilla JS
- [Swiper.js](https://swiperjs.com/) (CDN)
- GitHub Pages deployable

## Setup
1. Clone repo
2. Open `index.html` or deploy to GitHub Pages
3. Replace placeholder images in `/assets/img/`
